## mad_diagram_description

[mad_diagram](./mad_diagram.png)

Сложно читаемая диаграмма.  

Похоже на систему проверки статистических гипотез на экспериметальных данных.

Судя по всему система представлеят собой Command Line Interface (CLI) для запуска экспериментов.

Компонент **Validation** используется для валидации данных указанных в конфигурации.

Модуль **PySATL-experiment** для запуска и конфигурации эксперимента:
- Компонент **Configuration** - настройки для запуска эксперимента, можно указать гипотезу, криетрий, режим запуска и тд
- Компонент **Factory** - фабрика, отвечающая за создание экспериметов
- Компонент **Experiment** - компонент, внутри которого описана логика проведения эксперимента
- Компонент **Persistance** - компонент отчвечающий за хранение данных 
- Компонент **Generator** генерирующий данные для эксперимента
Модуль **PySATL-сriterion** для проверки соответствия данных различным статистическим тестам.
